# Getting Started

## Back End

### In the project directory of **BackEnd Api**(api), you can run:

## `npm start`

Server will run at port **3001**

---

### Dependencies:

You must run :

## `npm install`

for dependencies, in order the client to work !

## Front End

### In the project directory of **FrontEnd App**(client), you can run:

## `npm start`
